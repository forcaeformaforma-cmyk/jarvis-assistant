# JARVIS-Assistant 🤖

Um assistente de IA autônomo inspirado no JARVIS do Homem de Ferro, com capacidade de processamento de linguagem natural, integração com APIs externas, execução de tarefas automaticamente e aprendizado contínuo.

## Características

✨ **Processamento de Linguagem Natural em Português** - Entende e responde comandos em português natural

🔗 **Integração com APIs Externas** - Conecta-se a múltiplas APIs para expandir funcionalidades

⚙️ **Execução Automática de Tarefas** - Executa scripts e comandos automaticamente

🌐 **Interface Web & CLI** - Acesso via web (Dashboard) ou linha de comando

🧠 **Aprendizado Contínuo** - Aprende com interações anteriores e melhora respostas

## Instalação

```bash
git clone https://github.com/forcaeformaforma-cmyk/jarvis-assistant.git
cd jarvis-assistant
pip install -r requirements.txt
```

## Uso Rápido

### Interface de Linha de Comando
```bash
python main.py
```

### Interface Web
```bash
python main.py --web
```

## Estrutura do Projeto

```
jarvis-assistant/
├── core/                 # Núcleo do JARVIS
├── nlp/                  # Processamento de Linguagem Natural
├── apis/                 # Integração com APIs externas
├── tasks/                # Execução de tarefas
├── learning/             # Sistema de aprendizado
├── web/                  # Interface web
├── utils/                # Utilitários
├── config/               # Configurações
├── main.py              # Ponto de entrada
└── requirements.txt     # Dependências
```
